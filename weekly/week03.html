Things to Remember

  • Objects are mutable at any time
  • Object literal is preferred
  • can use compute property keys
  • Check if own properties exist before looping through
  • object properties order isn't ordered
  • Object literal to create namespaces
  • Math objects are subject to accuracy errors
  • Date params: new Date(year,month,day,hour,minutes,seconds,milliseconds)
  • RexEx are objects with properties
  • Greedy and Lazy Modifiers
  • DOM stores any whitespace text nodes.
  • Node Lists aren't actually arrays
