Things to Remember

  • Immediately Invoked Function Expressions: IIFE. Use to keep the global namespace clean and temporary variables.
    (function(){code here})();
    can be used with arrow function expression as well
    (()=>{code here})();
  • Lazy Definition pattern
  • Callback Hell. keep code clean and well organized. Promise helps with callback issues
