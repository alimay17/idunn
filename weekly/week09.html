Notes for Week 09

Things to Remember

  • Use global variables without the window prefix (window.x)
  • Browsers can disable dialogs. Use with caution
  • Don't rely on cookie expiry
  • Use css over JS for animation effects if possible

Research Needed

  • window.location properties
  • JS cookie libraries
  • data- attribute
  • Websockets
